# Zero to Code System

**非技术背景的人学 Coding，怎么把学到的东西系统地留下来？**

这是一个 Skill，帮你自动建立和维护一套结构化的编程学习文档。AI 自动帮你分类、整理、沉淀。
这个 Skill 来自我自己尝试探索 Coding 过程中沉淀的真实学习系统。我是产品背景、代码零基础，还在日常学习中继续迭代这套系统。

---
## 适合谁
- 产品经理 / 运营 / 设计师等非技术背景，试图探索 Coding
- 用 Cursor + AI 辅助学习和开发
- 希望学到的东西不散落、自动化积累、方便回顾

## 它解决什么问题

非技术背景学 Coding，最常见的困境：

- 学了不少，但知识散落在各个对话里，下次用的时候找不到
- 概念和操作混在一起，想查个命令要翻半天
- 每次开新对话，AI 不知道你之前学了什么，重复解释
- 没有系统的学习轨迹，不知道自己到底进步了多少

这个 Skill 的核心理念：**"怎么做"和"是什么"分开放，每天自动沉淀。**

## 它是什么

一套 5 个文件的学习文档体系 + 自动化的 AI 行为规则：

```
你的学习文档/
├── playbook.md           ← 实操手册（SOP + 提示词 + 命令速查）
├── knowledge.md          ← 知识库（概念按主题沉淀）
├── learning-log.md       ← 每日学习速记（最新在最上面）
├── open-questions.md     ← 待解决问题
└── projects/             ← 项目研究笔记
```

| 文件 | 存什么 | 什么时候翻 |
|------|--------|-----------|
| playbook.md | 操作流程、提示词、命令 | 做事时照着走 |
| knowledge.md | 概念定义、类比、关系图 | 碰到不懂的词回来查 |
| learning-log.md | 每天学到了什么 | 回顾学习轨迹 |
| open-questions.md | 还没搞懂的问题 | 下次学习前翻一翻 |
| projects/ | 研究过的项目 | 研究新项目时参考 |

## 核心能力

**自动内容分流** — 你说"记一下"，AI 自动判断这是 SOP、概念还是命令，写到对应文件

**收工自动日志** — 你说"收工"，AI 从对话中提取今天学到的重要知识点，确认后写入学习日志

**非技术友好** — 所有概念都配"一句话定义 + 生活类比"，命令都可以直接复制

**上下文预警** — 上下文快满时主动提醒你先保存，防止知识丢失

## 安装

### 1. 下载

```bash
git clone https://github.com/shuyan-5200/zero-to-code-system.git
```

### 2. 放到 Cursor Skills 目录

```bash
# 个人级（所有项目可用）
cp -r zero-to-code-system ~/.cursor/skills/

# 或项目级（只在当前项目可用）
cp -r zero-to-code-system .cursor/skills/
```

### 3. 开始使用

在 Cursor 里对 AI 说：

> "帮我建立学习文档系统"

AI 会自动读取 Skill，引导你完成文档初始化。

### 4. （可选）安装配套 Rule

Skill 按关键词触发，Rule 始终生效。安装配套 Rule 后可以获得：
- **身份协议** — AI 始终用非技术友好的方式和你沟通
- **自动日志** — 说"收工"自动触发日志沉淀
- **路径管理** — 学习文档放在 iCloud 等同步盘时自动处理软链接

Rule 模板见 [references/companion-rules.md](references/companion-rules.md)，按说明定制后保存到 `~/.cursor/rules/` 即可。

## 使用示例

**学到新概念时：**
> "把刚才讲的xx概念整理到学习手册里"
>
> AI 自动写入 knowledge.md，格式：名称 | 一句话定义 | 生活类比

**学到新操作流程时：**
> "帮我把这个xx流程整理成 SOP"
>
> AI 写入 playbook.md，带可复制的命令和速查卡片

**结束学习时：**
> "收工"
>
> AI 列出今天学到的知识点 → 你确认 → 自动写入 learning-log.md

**有未解决的问题：**
> "这个问题先记着，下次再研究"
>
> AI 写入 open-questions.md，下次可以直接翻
